# @wiki3-ai/webllm-chat-kernel

WebLLM chat kernel for JupyterLite using @mlc-ai/web-llm.

## Installation

```bash
pip install webllm-chat-kernel
```

Or for development:

```bash
pip install -e .
```

## Development

```bash
# Install npm dependencies (updates package-lock.json)
npm install

# Build the extension
npm run build

# Install in development mode
pip install -e .
```
