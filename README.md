# @wiki3-ai/webllm-chat-kernel

WebLLM chat kernel for JupyterLite using @built-in-ai/web-llm.

## Installation

```bash
pip install webllm-chat-kernel
```

Or for development:

```bash
cd lite-kernel
pip install -e .
```

## Development

```bash
# Install npm dependencies
npm install

# Build the extension
npm run build

# Install in development mode
pip install -e .
```
